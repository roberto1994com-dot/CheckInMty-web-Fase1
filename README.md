# CheckInMty:

Página web informativa para la reservación de cabañas en la zona de Monterrey, Nuevo León.

## Objetivo
Ofrecer una plataforma confiable donde los alojamientos son verificados físicamente.

## Tecnologías
- HTML
- CSS
- JavaScript

## Metodología
- DevOps
- Estrategia de ramificación: GitHub Flow
